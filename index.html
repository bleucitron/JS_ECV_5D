Difference with classic JS


// classic JS
document.getElementById('first').innerHTML = 'HEY';
document.getElementById('first').style.color = 'red';

// jQuery
$("#first").html('HEY');
$("#first").css('color', 'red');

Refining selectors


$("div.myClass").has("p");     // div.myClass elements that contain p tags
$("h1").not(".blue");          // h1 elements that don't have a class of bar
$("ul li").filter(".current"); // unordered list items with class of current
$("ul li").first();              // just the first
$("ul li").eq(5);              // just the sixth
